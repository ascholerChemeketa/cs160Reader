Parallel Processing
==============================

This video introduces parallel processing in BYOB.

.. youtube:: INYI9CYXEaQ
    :height: 315
    :width: 560

You can try out the programs you see in the video using the scripts available from the starter programs zip file.

.. note::

  BYOB doesn't really allow multiple things to happen at once. It simulates multiple things happening be switching rapidly between tasks.
