Little Computer Assembler

Input code into the left box. Then click "Assemble Code". The Result pane will show the output after you run the assembler. If all goes well, you will see your final code output after a line like this: "--------------------". You can copy and paste those lines (not including the "---------------" or anything above it) directly into the Little Computer memory to test.
There is an assembly language reference at the bottom or side of the screen.
